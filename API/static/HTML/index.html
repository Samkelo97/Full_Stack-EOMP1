Router description

HTTP Methods Path Description
GET / (Home page) Directs you to the landing page
GET /users gets all the users
GET /user/:id gets a single user by id
POST /user Registers a single user
PATCH/PUT /user/:id Updates a single user by id
DELETE /user/:id Removes a single user by id
GET /products gets all the products
GET /product/:id gets a single product by id
POST /product creates a new product
PUT/PATCH /product/:id Updates a single product
DELETE /product/:id Removes a product
